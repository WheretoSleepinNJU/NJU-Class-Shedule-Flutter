# 外观与主题 操作逻辑说明与PRD

## 1. 文档目标
- 定义“外观与主题”二级页面的模式切换与背景图片管理逻辑。
- 明确背景图对课程表主页面的视觉替换行为。

## 2. 页面定位
- 页面名称：外观与主题
- 入口：我的主页面 -> 外观与主题
- 页面层级：二级页面

## 3. 顶栏规范
- 左上：返回键
- 中间：标题“外观与主题”（顶栏居中）

## 4. 模式选择逻辑
- 可选项：浅色模式、深色模式
- 选中态：浅黄容器高亮
- 非选中态：白色容器
- 点击行为：
  - 点击某模式后立即切换 `themeMode`
  - UI 实时应用对应主题变量

## 5. 背景图片逻辑

### 5.1 上传背景图片
- 点击“上传背景图片”
- 拉起系统相册/文件选择器
- 选图成功后：
  - 保存背景图配置
  - 替换课程表主页面默认白色大底

### 5.2 清除背景图片
- 点击“清除背景图片”
- 清空背景图配置
- 课程表主页面恢复默认白色背景

## 6. 事件定义
- `onBackFromTheme()`
- `onSelectTheme(mode)`
- `onUploadBackgroundImage()`
- `onApplyBackgroundImage(url)`
- `onClearBackgroundImage()`

## 7. 验收标准
- 顶栏标题居中。
- 浅/深色模式可切换，且选中态高亮正确。
- 上传背景图片后课程表主页面背景被替换。
- 清除背景图片后课程表主页面恢复默认背景。
